БАЗОВЫЕ

- Регистрация через логин (+)
- Логин обязательный (+)
- Выход /logout (+)
- Хеширование паролей (+)
- Сессии (+)
- Необходимый интерфейс
- Минимальный CSS для удобства


АДМИН

- Админ-панель
- Админ видит список пользователей
- Админ может удалять пользователей
- Админ может создавать пользователей
- Админ может отключить авто регистрацию
- Только админ видит админ-кнопки
- Просмотр базы данных


СООБЩЕНИЯ С ИЗОБРАЖЕНИЯМИ

- Сообщение может содержать 1 ссылку на изображение
- Ссылка сохраняется в БД
- Если есть ссылка — картинка отображается
- Если нет — просто текст
- Нельзя прикрепить больше 1 изображения


СИСТЕМА ОТДЕЛЬНЫХ ЧАТОВ

- Чаты можно создавать на /main
- У чата есть:
    - имя
    - пароль
- Чат защищён паролем
- Пользователь должен "вступить" в чат
- После вступления чат запоминается
- /main показывает список чатов пользователя
- Каждый чат открывается отдельно
- Сообщения принадлежат конкретному чату
